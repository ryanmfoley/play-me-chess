# Play-Me-Chess

## Description:

<strong>Play-Me-Chess</strong> was made so that users could play chess with their friends.

## Technologies Used:

**<u>Back-end</u>**

- NodeJS
- Express
- MongoDB
- Mongoose
- Socket.io

**<u>Front-end</u>**

- JavaScript
- Socket.io
- HTML
- CSS
- Bootstrap

## Installation:

Open the deployed app at https://play-me-chess.herokuapp.com. Feel free to fork and clone this repository.
