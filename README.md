<h3>Play-Me-Chess</h3>

Description:

Play-Me-Chess was made so that users could play chess with their friends.

Technologies Used:

<u>Back-end</u>

<ul>
  <li>NodeJS</li>
  <li>Express</li>
  <li>MongoDB</li>
  <li>Mongoose</li>
  <li>Socket.io</li>
</ul>

<u>Front-end</u>

<ul>
  <li>JavaScript</li>
  <li>Socket.io</li>
  <li>HTML</li>
  <li>CSS</li>
  <li>Bootstrap</li>
</ul>

Installation:

Open the deployed app at https://play-me-chess.herokuapp.com. Feel free to fork and clone this repository.
